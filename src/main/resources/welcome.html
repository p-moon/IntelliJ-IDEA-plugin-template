欢迎使用 Call Graph ++

配置使用 MCP Server：


{
  "mcpServers": {
    "code-analysis": {
      "autoApprove": [
        "method-json-call-stack",
        "method-svg-call-stack"
      ],
      "disabled": false,
      "timeout": 10000,
      "url": "http://127.0.0.1:${PORT}/sse",
      "transportType": "sse"
    }
  }
}

分析某一个函数的调用行为：

该插件会根据 AST 的代码树自动查找接口实现来分析业务逻辑。

数据库相关的查询和写入操作会被高亮出来。如果你想要批量分析，可以选中某个文件或文件夹，将会分析所有选中内容下的所有的函数的调用行为
